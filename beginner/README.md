# Beginner projects

Small, focused projects for people who know basic Python syntax and want to
build something complete from start to finish.

## What "beginner" means here

- Uses the Python standard library wherever it keeps the lesson clear.
- Teaches one or two core ideas at a time (files, data structures, a simple
  CLI, input validation).
- Small enough to read in one sitting, but still tested and runnable.

**You should be comfortable with:** variables, functions, loops, conditionals,
and running a script from the terminal.

## Projects

Projects are added incrementally. See the [roadmap](../docs/roadmap.md) for what
is planned, and the [learning paths](../docs/learning-paths.md) for a suggested
order.

_No projects are published in this level yet._

## How this level is organised

Each project lives in its own folder named `NN-project-slug` (for example
`01-expense-tracker-cli`) and includes its own README, tests, and run
instructions. See the [repository map](../docs/repository-map.md) for the
naming rules and the difference between a draft and a published project.
