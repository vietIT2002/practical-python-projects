**Practical Python Projects v0.1.0 is out.**

It's an open-source collection of complete, tested Python projects for people who
want to learn by building — each one runnable from documented commands and backed
by real tests and CI.

This first release includes three launch projects:

- **Expense Tracker CLI** (beginner) — exact money handling with `Decimal` and
  safe, atomic storage.
- **Safe File Organizer CLI** (beginner) — organise files with a dry-run
  default, no overwrites, and undo.
- **URL Shortener API** (intermediate) — FastAPI + SQLAlchemy with migrations
  and isolated tests.

Plus the foundation: a `uv` toolchain, CI on Python 3.12–3.14, a project
template with validated metadata, and generated project catalogs.

These are learning projects, not production software, and each is honest about
its limitations.

Repository: https://github.com/vietIT2002/practical-python-projects

Feedback and contributions are welcome.
