# Release notes — v0.1.0

The first public release of **Practical Python Projects**: a solid repository
foundation and three complete, tested launch projects.

## What's included

- A modern, reproducible toolchain (`uv`, Ruff, mypy, pytest, pre-commit) and
  continuous integration across Python 3.12–3.14.
- Community, security, and contribution files, plus issue and pull request
  templates.
- A project template and a machine-validated metadata contract, with project
  catalogs generated from that metadata.
- Three launch projects:
  - [Expense Tracker CLI](../../beginner/01-expense-tracker/README.md) — a
    dependency-free CLI with exact `Decimal` money handling and atomic storage.
  - [Safe File Organizer CLI](../../beginner/02-file-organizer/README.md) — a
    file organiser with a dry-run default, no overwrites, and undo.
  - [URL Shortener API](../../intermediate/01-url-shortener-api/README.md) — a
    FastAPI service with SQLAlchemy, Alembic migrations, and isolated tests.

## Getting started

From a clean clone you need [Python 3.12+](https://www.python.org/downloads/),
[uv](https://docs.astral.sh/uv/), and git:

```sh
uv sync --group dev
uv run pytest
```

Each project has its own README with run and test instructions. See the
[documentation home](../README.md) and the
[project catalog](../project-catalog.md).

## Known limitations

- These are **learning projects**, not production-hardened software.
- The URL Shortener API is an intentional open redirector with **no rate
  limiting or authentication**; read its
  [threat model](../../intermediate/01-url-shortener-api/docs/threat-model.md)
  before deploying anything based on it.
- Click counts in the URL shortener are approximate.

This release is **not** described as stable for production use.
