# Getting support

Need help using the projects or setting up your environment? Here is where to
go.

## Questions and help

- Read the [documentation](docs/README.md), starting with
  [getting started](docs/getting-started/README.md).
- Each project has its own README with run and troubleshooting instructions.
- For general questions, open a **Question / help** discussion (once GitHub
  Discussions is enabled) or a documentation issue.

## Reporting a problem

- Something is broken? Open a **Bug report** issue.
- Documentation unclear or wrong? Open a **Documentation** issue.
- Want a new capability or project? Open a **Feature request** or
  **New project proposal** issue.

## Security issues

Do not use public channels for security problems. Follow the private reporting
route in [SECURITY.md](SECURITY.md).

## Please note

This is a community, educational project maintained on a best-effort basis.
There is no commercial support or guaranteed response time.
