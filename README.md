# Practical Python Projects

Learn Python by building practical, tested, and production-minded projects.

> Status: early foundation. The repository toolchain is being established
> incrementally. Launch projects and the full landing page arrive in later
> stages; see the [roadmap](docs/roadmap.md).

## Repository model

This repository is a curated collection of independent Python learning
projects. The root owns shared development tooling and quality gates; each
project is self-contained and runnable on its own. See the architecture
decisions in [`docs/architecture/`](docs/architecture/) and the plan in
[`docs/roadmap.md`](docs/roadmap.md).
